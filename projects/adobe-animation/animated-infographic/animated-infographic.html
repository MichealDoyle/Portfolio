Animated Infographic

The aim of this project is the create an animated infographic explaining a simple concept or process. The goal of this project is to learn how to use the tools Animate offers along with design principles to create a smooth flowing and engaging infographic that displays the intended information easily.

Tools Used

  • Text Tool
  • Classic Tween
  • Motion Tween
  • Shape Tween
  • Actionscript
  • Symbols

Skills Displayed

  • Editing With Tweens
  • Transitioning
  • Creating Buttons
  • Adding Interactivity
  • Typography

Reflections

The outcome of this project is a fun and interactive infographic about how to make a grilled cheese sandwich. This project taught me how to use transitions and tweens to switch "slides", how to to use shape tweens to make lines appear as if theyre growing or shrinking, and how to use transitions and tweens to hide and reveal information to make the content more engaging to follow along with.
