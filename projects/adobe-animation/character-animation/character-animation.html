Character Animation

This project required us to create an animation of a character doing a simple task. The goal of this project was to create an animation that focused on basic movement, timing, and character design.

Tools Used

  • Fluid Brush Tool
  • Motion Tween
  • Camera Tween
  • Symbols

Skills Displayed

  • Chacter Design
  • Timing
  • Character Movement
  • Editing With Tweens

Reflection

The outcome of this project is a very simple and silly character walking down a path to say hello to you at the end of the road. This project helped me learn how to take a group of simple shapes and turn them into a fun cartoon character using tweens and other animation principles.
