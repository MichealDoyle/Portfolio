Interactive Web Banner

This project required us to create an interactive web banner that responded to user actions. The goal was to practice layouts and designs for web banners, while also adding interactivity to them with button symbols.

Tools Used

  • Text Tool
  • Motion Tween
  • Classic Tween
  • Primitive Oval Tool
  • Primitive Rectangle Tool
  • Symbols

Skills Displayed

  • Adding Interactivity
  • Creating Buttons
  • Layout Composition
  • Typography

Reflection

This project helped me learn how to use animation along with design to create a nice interactive web banner for a fake travel company. I became more comfortable working with tweens and symbols.
