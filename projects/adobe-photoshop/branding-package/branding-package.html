Branding Package for Fictional Start-up

This project consisted of designing a branding package for a fictional startup, complete with a logo, business card, and social media banner. The overall goal of the project was to create brand identity that was consistent, clean, and visually appealing, displayed through multiple ways of marketing.

Tools Used

  • Shape tools
  • Text tools
  • Smart filters
  • Color overlay
  • Gradient overlay
  • Layer styles

Skills Displayed

  • Logo design
  • Layout
  • Color scheme
  • Marketing consistency
  • Typography

Reflection

The final outcome of the project is a very simple, smooth, and visually pleasing branding package that displays the theme of the fictional company through multiple forms of marketing techniques. The simple logo is easily brandable and offers nice color contrast to help pop to the eye and stand out. The font choices help slightly with the aesthetics and readability throughout the multiple documents. The simple black and white color scheme of the logo and using the shades of green to help with the billiards theme offer a good color contrast as visual appeal and readability.
